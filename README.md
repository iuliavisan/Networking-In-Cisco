# Networking-In-Cisco

## English Version

### Overview
This repository serves as a centralized hub for my computer networking expertise and ongoing professional development. It integrates academic projects developed during university courses with structured study materials for the **Cisco Certified Network Associate (CCNA)** certification.

### Repository Structure
* **Networking-Projects-University**: Contains practical network designs and simulations implemented in Cisco Packet Tracer.
* **CCNA-Preparation**: Dedicated to certification study notes, CLI commands, and network theory fundamentals.

### Core Competencies
* **Network Design**: Architecting LAN/WAN environments with hierarchical models.
* **Cisco IOS Administration**: Hands-on experience with router and switch configuration via CLI.
* **Service Implementation**: Configuring DNS, DHCP, FTP, and Web services within virtualized environments.
* **Security & Filtering**: Implementing Access Control Lists (ACLs) and basic device hardening.

## Versiunea în Română

### Prezentare Generală
Acest repository funcționează ca un hub centralizat pentru expertiza mea în rețele de calculatoare și dezvoltarea profesională continuă. Acesta integrează proiecte practice dezvoltate în cadrul cursurilor universitare cu materiale de studiu structurate pentru certificarea **Cisco Certified Network Associate (CCNA)**.

### Structura Repository-ului
* **Networking-Projects-University**: Conține design-uri de rețea practice și simulări implementate în Cisco Packet Tracer.
* **CCNA-Preparation**: Folder dedicat notițelor de studiu pentru certificare, comenzilor CLI și fundamentelor teoretice de rețea.

### Competențe Centrale
* **Design de Rețea**: Arhitecturarea mediilor LAN/WAN folosind modele ierarhice.
* **Administrare Cisco IOS**: Experiență practică în configurarea routerelor și switch-urilor prin interfața CLI.
* **Implementare Servicii**: Configurarea serviciilor DNS, DHCP, FTP și Web în medii virtualizate.
* **Securitate și Filtrare**: Implementarea Listelor de Control al Accesului (ACL) și securizarea de bază a dispozitivelor.
